# csdwheels

日常工作中经常会发现有大量业务逻辑是重复的，而用别人的插件和轮子也不能完美解决一些定制化的需求，所以我抽取出来了这套插件库，希望能让大家提升工作效率，少加班~

> 项目插件使用 ES5 语法及 UMD 规范封装，部分插件提供 ES6 及 Vue 版本（ES5版本只支持 script 标签的引入方式，ES6版本支持直接 import 导入）
